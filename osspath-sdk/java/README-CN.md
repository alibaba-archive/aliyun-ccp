## Alibaba Cloud OSSPath SDK for Java


## 安装

```xml
<dependency>
    <groupId>com.aliyun</groupId>
    <artifactId>ccp-baseclient</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
<dependency>
    <groupId>com.aliyun</groupId>
    <artifactId>ccp-oss-client</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
```
