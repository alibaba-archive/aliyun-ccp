## Alibaba Cloud OSSPath SDK for TypeScript/Node.js

## 要求

Node.js >= 8.x

## 安装

你可以使用 `npm` 来管理你的依赖，并通过下面命令安装依赖并写入package.json

```sh
$ npm install @alicloud/osspath-sdk -S
```
