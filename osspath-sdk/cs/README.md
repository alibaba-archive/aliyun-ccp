## Alibaba Cloud OSSPath SDK for .Net

## Installation

Use .Net CLI ( Recommand )

    dotnet add package Aliyun.SDK.CCP.OSSClient

Use Package Manager

    Install-Package Aliyun.SDK.CCP.OSSClient
