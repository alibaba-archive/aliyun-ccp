## Alibaba Cloud CCP BaseClient for .Net


## 安装

Use .NET CLI ( Recommand )

    dotnet add package Aliyun.SDK.CCP.BaseClient

Use Package Manager

    Install-Package Aliyun.SDK.CCP.BaseClient
