## 安装

- **使用 pip 安装(推荐)**

如未安装 `pip`, 请先至pip官网 [pip user guide](https://pip.pypa.io/en/stable/installing/?spm=5176.doc53090.2.7.zHDiNV "pip User Guide") 安装pip .

```bash
# 安装 ccp-path-sdk
pip install alibabacloud_ccppath_sdk
```
